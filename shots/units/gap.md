**description:** Wrapping flex with 1rem row gap and 2rem column gap.
**csss:** $flex(wrap,gap(1rem,2rem))
**css:**
```css
.\$flex\(wrap\,gap\(1rem\,2rem\)\) {
  flex-wrap: wrap;
  gap: 1rem 2rem;
}
```
