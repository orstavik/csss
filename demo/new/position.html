Position

$relative $absolute $fixed

$relative — with inset

position: relative; optional start/end/top/bottom offsets.

$relative(start, 1rem, 5%) — offset from start/top

$absolute — inside a relative parent

position: absolute; top/right/bottom/left or logical insets.

$absolute(rightTop, 0.5rem, 0.5rem)
$absolute(leftBottom, 0.5rem, 0.5rem)

$fixed

position: fixed; e.g. rightBottom(0,0) for a corner badge.

Fixed elements use $fixed(…) — e.g. $fixed(rightBottom, 0, 0).
