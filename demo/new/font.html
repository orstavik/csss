Font

$Typeface $font $Font Web fonts

$Typeface — umbrella definition

Define a named typeface (family, size, weight, etc.) for reuse with $Font(name).

$Typeface(main, Inter, sans-serif, 16px, 400)

This body uses the main typeface. Headings use heading.

$font — direct properties

Set font-family, size, weight, style, etc. directly.

Plain text, $font(700) bold, $font(italic) italic.

$Font(name) — apply umbrella

Apply a typeface defined with $Typeface by name.

$Font(heading) — heading typeface

$Font(main) — main typeface. Droplets like $font(600) override within scope.

Web fonts & @font-face settings

Load fonts from URLs and set @font-face descriptors (family, weight, style, stretch, featureSettings) via URL hash.

$Font(display) — Roboto from CDN (lg, bold)
$Font(title) — ComicDisplay (#family=&weight=700&style=italic)
$Font(caption) — ComicCaption (#family=&stretch=condensed)
