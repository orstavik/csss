Typography: Cloud & Umbrella & Droplet Demo

This page demonstrates the CSSS font system. The `body` tag defines several "umbrella" `$Typeface()`s. Child elements then use `$font()` to apply an umbrella and can add "droplets" such as `$fontStyle()` to override specific properties within the `$font()` scope.

Transitional

font-family: 'Charter', 'Bitstream Charter', 'Sitka Text', Cambria, serif;

This is the base transitional font. It's a classic, highly readable serif, great for long-form text. It inherits all properties from the `transitional` umbrella.

This bold paragraph uses a droplet 300 to increase the font weight to 600, making it bolder than the base text, with an even bolder strong tag.

This paragraph uses an italic font, which is reset to normal inside this quote: "normal style here"

Neo-Grotesque

font-family: 'Inter', 'Roboto', 'Helvetica Neue', Arial, sans-serif;

This is the base neo-grotesque font. It's a modern, clean, and versatile sans-serif, perfect for UI and general use.

This paragraph uses a droplet to set a lighter font weight (300), while the span tag inside it is made heavier (600).

This paragraph uses the `$font(smaller,bolder)` keyword droplets.

Monospace Code

font-family: 'ui-monospace', 'Cascadia Code', 'Source Code Pro', monospace;

This is the base monospace font, ideal for div snippets and technical text, such as:

const umbrella = "$Typeface(monoCode, ...)";

This paragraph has `$font(noKerning)` via a droplet.

Old Style

font-family: 'Iowan Old Style', 'Palatino Linotype', 'URW Palladio L', P052, serif;

This is an "Old Style" font. These typefaces have a classic, distinguished look, often used in books and formal documents for their excellent readability.

This paragraph uses the `smaller` keyword to reduce the font size, making it suitable for captions or footnotes.

Industrial

font-family: Bahnschrift, 'DIN Alternate', 'Franklin Gothic Medium', 'Nimbus Sans Narrow', sans-serif-condensed, sans-serif;

This is an "Industrial" sans-serif font. It features a condensed and efficient design, often seen in signage and user interfaces where space is limited.

This paragraph uses droplets to apply an `$font(expanded)` font stretch, showing how a condensed font can be made wider.

Comic Neue

font-family: 'MS Comic Sans', var(--comicFontFamily, unset);

This is the Comic Neue font, a modern take on the classic Comic Sans. It's more refined and versatile, suitable for informal documents and presentations.

This paragraph uses the `$font(italic)` droplet keyword as droplets to enhance its appearance.
